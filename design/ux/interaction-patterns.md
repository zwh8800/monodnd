# 交互模式库 (Interaction Pattern Library)

> **状态**: 草稿 (Draft) — 启动于 Technical Setup → Pre-Production 门控检查
> **版本**: v0.1
> **适用范围**: 全部 UI 屏幕
> **关联文档**: `design/art/art-bible.md` §7, `src/DndGame/UI/PixelTheme.cs`, `design/ux/hud.md`

---

## 1. 概述

本文档定义《酒馆与命运》中跨屏幕共享的通用交互模式。任何新增 UI 屏幕必须：

1. 优先使用本文档中已有的交互模式
2. 如需新模式，先在此文档登记（附使用场景和理由）
3. 不引入与现有模式冲突的交互行为

---

## 2. 模式 1: 确认/取消对话框

### 使用场景
- 删除存档、解散角色、覆盖文件等**不可逆操作**
- LLM API 调用失败时的**错误提示**

### 视觉规范
```
┌─────────────────────────────────┐
│  ⚠ 确认解散 [角色名]？          │ ← 标题（18px, 白色）
│                                  │
│  该角色将永久离开队伍，           │ ← 描述（14px, SecondaryText）
│  此操作不可撤销。                 │
│                                  │
│  ┌──────────┐ ┌──────────┐      │
│  │   确认   │ │   取消   │      │ ← 按钮（PixelTheme.ButtonStyle）
│  └──────────┘ └──────────┘      │
└─────────────────────────────────┘
```

### 交互规则
| 输入 | 行为 |
|------|------|
| `Enter` | 执行确认操作（默认焦点在"确认"按钮） |
| `Esc` | 执行取消操作，关闭对话框 |
| `← →` | 在按钮间切换焦点 |
| `Tab` | 焦点在按钮间循环 |

### 键盘快捷约定
- **确认（危险操作）**: `Enter` 不可直接触发 —— 需先 `→` 将焦点移到"确认"按钮再 `Enter`
- **确认（安全操作）**: `Enter` 可直接触发（默认焦点在确认按钮）
- **取消**: 始终 `Esc`

### 无障碍
- 焦点状态用 2px 边框高亮（`ButtonHoverBorder` 颜色）
- 确认/取消按钮**永远不**仅靠颜色区分（文字本身就是区分）

## 3. 模式 2: 列表选择

### 使用场景
- 角色招募列表（酒馆）
- 物品栏浏览（战斗/酒馆）
- 法术书选择
- 存档槽位选择

### 视觉规范
```
┌─────────────────────────────────┐
│  冒险者招募                      │ ← 标题
│ ┌───────────────────────────┐   │
│ │ ▶ 卡尔·石拳  Lv3 战士    │   │ ← 选中项（浅蓝背景）
│ │   艾琳·星歌  Lv2 法师    │   │ ← 未选中项
│ │   格雷戈·铁盾 Lv1 牧师   │   │
│ │   ...                    │   │
│ └───────────────────────────┘   │
│                                  │
│  [选中角色详细信息预览区域]       │ ← 右侧/下方预览
└─────────────────────────────────┘
```

### 交互规则
| 输入 | 行为 |
|------|------|
| `↑ ↓` | 在列表项间导航 |
| `Enter` | 确认选择当前项 |
| `Tab` | 切换到详情预览面板（若存在） |
| `Esc` | 退出列表，返回上级菜单 |
| `PageUp/Down` | 翻页（列表 > 10 项时） |

### 列表项显示约定
- 每项 ≥ 36px 高度（1280×720, 24px 字体 + 12px 间距）
- 选中项：浅蓝半透明背景 + 左侧 `▶` 三角标记
- 不可选项：灰色文字 (`HintText` 色) + 不可聚焦
- 列表超过可视区域时，右侧显示滚动条（8px 宽）

## 4. 模式 3: 数值增减

### 使用场景
- 属性点分配（创建角色）
- 物品数量调整（购买/出售/使用）
- 设置滑块（音量、亮度等）

### 视觉规范
```
┌──────────────────────────────┐
│  力量:  [◀]  15  [▶]       │ ← 左右箭头 ± 文字显示
│  敏捷:  [◀]  12  [▶]       │
│  剩余点数: 3                │
└──────────────────────────────┘
```

### 交互规则
| 输入 | 行为 |
|------|------|
| `← →` | 选择属性行 |
| `↑ ↓` | 在当前属性上增减（若可行） |
| `+ / -` | 增减当前选中属性值 |
| `Enter` | 确认分配 |
| `Esc` | 取消，恢复原值 |

### 视觉反馈
- 达到上限时 `[▶]` 箭头变灰
- 达到下限时 `[◀]` 箭头变灰
- 数值变化时短暂闪烁（100ms, 白色→恢复）

## 5. 模式 4: 标签页/分区导航

### 使用场景
- 角色属性面板（属性/装备/法术/关系 4 标签）
- 设置界面（画面/声音/控制/游戏 4 标签）
- 酒馆主界面（招募/任务/商店/日志 4 标签）

### 视觉规范
```
┌─────────────────────────────────┐
│  ┌──────┬──────┬──────┬──────┐  │
│  │ ▶属性│ 装备 │ 法术 │ 关系 │  │ ← 标签栏
│  └──────┴──────┴──────┴──────┘  │
│  ┌──────────────────────────┐   │
│  │                          │   │
│  │  当前标签页内容           │   │
│  │                          │   │
│  └──────────────────────────┘   │
└─────────────────────────────────┘
```

### 交互规则
| 输入 | 行为 |
|------|------|
| `← →` | 切换标签页 |
| `Tab` | 焦点进入标签页内容区域 |
| `Shift+Tab` | 焦点回到标签栏 |
| `Esc` | 关闭面板 |

### 标签设计约定
- 选中标签：亮色背景 + 下划线（2px）
- 未选中标签：暗色背景
- 标签宽度自适应文字长度（最少 80px）
- 最多 6 个标签（超出用 `...` + 下拉）

## 6. 模式 5: 通知/Toast

### 使用场景
- 战斗结果通知（"对地精造成 8 点伤害"）
- 系统通知（"存档成功"）
- LLM 状态通知（"AI 叙述生成中..."）
- 物品获取（"获得 铁剑 ×1"）

### 视觉规范
```
┌────────────────────────────┐
│ 🗡 对地精造成 8 点伤害     │ ← 半透明暗底 + 白色文字
└────────────────────────────┘
         ↑ 屏幕上方居中，向下偏移 80px
```

### 交互规则
| 属性 | 值 |
|------|-----|
| 显示时长 | 2.5 秒（自动消失） |
| 最大同时显示 | 3 条（FIFO 队列） |
| 堆叠方向 | 从上往下，间距 8px |
| 动画 | 淡入 200ms, 淡出 300ms |
| 可中断 | 否 —— 通知为纯信息展示，不接受输入 |

### 通知类型与颜色

| 类型 | 图标 | 边框色 | 示例 |
|------|:--:|--------|------|
| 伤害 | 🗡 | `DamageRed` | "对地精造成 8 点伤害" |
| 治疗 | ✚ | `HealingGreen` | "回复 5 点生命值" |
| 获得物品 | 📦 | `Gold` | "获得 治疗药水" |
| 系统信息 | ℹ | `SecondaryText` | "存档成功" |
| 错误 | ⚠ | `DamageRed` | "连接失败，使用离线模式" |
| LLM 状态 | 🤖 | `HintText` | "AI 正在生成叙述..." |

## 7. 全局交互约定

### 焦点管理
- 每个屏幕有**明确的初始焦点**（通常是第一个可交互元素）
- `Tab` 和 `Shift+Tab` 在所有可交互元素间循环
- 焦点切换遵循**视觉阅读顺序**（从上到下，从左到右）
- 不可交互元素永不获取焦点

### 键盘快捷
| 全局快捷键 | 行为 | 适用范围 |
|-----------|------|---------|
| `Esc` | 返回上一级 / 关闭当前面板 | 全局 |
| `Enter` | 确认当前选择 | 全局 |
| `Tab` | 焦点移至下一个元素 | 全局 |
| `Shift+Tab` | 焦点移至上一个元素 | 全局 |
| `F1` | 显示当前上下文帮助 | 全局（可选） |
| `F5` | 快速存档 | 非战斗场景 |
| `F9` | 快速读档 | 非战斗场景 |

### 像素主题
所有 UI 元素共享 `PixelTheme.cs` 定义的样式。新增控件不得创建临时颜色值——必须从 `PixelTheme` 静态常量引用。
