# Quick Design Spec: 设置选项 (Settings/Options)

**Type**: New Small System
**Scope**: 提供音量控制、键位配置和语言选择的设置界面。独立 Meta 系统，不依赖任何游戏玩法系统。
**Date**: 2026-05-09

---

## Overview

设置选项系统让玩家调整游戏的视听体验和操作方式。这是一个独立的 Meta 系统——它不参与游戏循环，纯粹为玩家提供控制权。MVP 阶段提供音量控制和基本键位配置。

---

## Core Rules

### 音量设置

三组独立音量滑块：
- **主音量**：控制所有声音的总体输出（0-100%）
- **音效音量**：骰子音效、战斗音效（0-100%，受主音量约束）
- **音乐音量**：BGM（MVP 阶段不可用，灰色显示，标注"Phase 2"）

### 键位设置

MVP 阶段显示当前键位映射表（只读），暂不支持自定义改键。Phase 2 支持改键。

默认键盘映射：

| 按键 | 功能 | 场景 |
|------|------|------|
| Enter | 确认/进入酒馆 | 主菜单 |
| R | 打开招募板 | 酒馆 |
| Q | 打开任务板 | 酒馆 |
| P | 打开队伍管理 | 酒馆 |
| E | 出发冒险 | 酒馆 |
| Esc | 返回/退出 | 全部 |
| 1-4 | 对话选项选择 | 对话 |
| H | 对话历史 | 对话 |

### 语言设置

MVP 仅支持简体中文。语言选项显示"简体中文"并置灰（不可切换）。Phase 2 添加英文。

### 访问方式

- 主菜单：设置按钮进入
- 酒馆：按 Esc 打开菜单，选择"设置"

## Acceptance Criteria

- [ ] **AC-SET.01**：主菜单中可进入设置界面，按 Esc 返回
- [ ] **AC-SET.02**：音量滑块可拖动调整，音效实时反映变化
- [ ] **AC-SET.03**：键位映射表正确显示当前所有按键功能
- [ ] **AC-SET.04**：设置关闭后变更持久保存（不随游戏重启丢失）
