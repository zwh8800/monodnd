# Quick Design Spec: 音频系统 (Audio System)

**Type**: New Small System
**Scope**: 管理游戏的音效播放和背景音乐切换。MVP 阶段仅覆盖骰子音效和基础战斗音效——BGM 和环境音属 Phase 2。
**Date**: 2026-05-09

---

## Overview

音频系统是 Pillar 2（战术深度）的感官锚点。骰子落地的声音让玩家感受到"我在玩桌游 DND"——d20 滚动的物理感通过音效传达。自然 20 的清脆提示音和自然 1 的低沉闷响，让玩家不需要看屏幕就知道"投出了什么"。

MVP 范围：最小可行音效集——骰子 + 战斗。不做 BGM，不做环境音，不做语音。

---

## Core Rules

### MVP 音效清单

**骰子音效**（最高优先级）：

| 音效 | 触发时机 | 听觉特征 |
|------|---------|---------|
| 骰子滚动 | 每次掷骰开始 | 短促的咔嗒滚动声，0.3 秒 |
| 骰子落地 | 掷骰结果确定 | 厚重落地声，0.2 秒 |
| 自然 20 | 投出暴击 | 清脆金属高音 + 短和弦，0.5 秒 |
| 自然 1 | 投出大失败 | 低沉闷响 + 破碎声，0.5 秒 |

**战斗音效**：

| 音效 | 触发时机 | 听觉特征 |
|------|---------|---------|
| 物理攻击命中 | 近战/远程攻击命中 | 砍击/穿刺短音，8bit 风格 |
| 物理攻击未命中 | 近战/远程攻击失手 | 破空声 + 轻微"叮"（弹开），8bit |
| 法术施放 | 施法动作开始 | 魔法嗡鸣渐强，0.3-0.5 秒（不同学派有变化） |
| 暴击伤害 | 暴击触发（攻击端） | 沉重打击 + 短促高音反馈 |
| 角色受伤 | 受到伤害 | 短促痛呼/盔甲碰撞 |
| 敌人死亡 | 敌人 HP 归零 | 破碎/消散声 |

### 音效风格

- **8bit/chiptune 风格**——像素艺术的听觉对应物
- 简洁、有辨识度、不冗长
- 自然 20 的音效应该让人**满足**——像街机游戏的高分音效
- 战斗音效每段 0.1-0.3 秒——不打断回合制节奏

### 音量控制

音频系统读取设置选项系统中的音量值：
- 主音量 × 音效音量 = 实际播放音量
- 音量变化立即生效，无需重启

### Phase 2 扩展

- BGM：酒馆（温馨）、冒险地图（神秘）、战斗（紧张）三套循环
- 环境音：酒馆背景（炉火噼啪/酒杯碰撞）
- 角色语音：受击/死亡短呼

## Acceptance Criteria

- [ ] **AC-AUD.01**：掷骰时播放滚动音效 + 落地音效，两个音效连续不重叠
- [ ] **AC-AUD.02**：自然 20 播放特殊提示音，与普通落地音效不同且可辨识
- [ ] **AC-AUD.03**：自然 1 播放特殊提示音，与普通落地音效不同且可辨识
- [ ] **AC-AUD.04**：战斗攻击命中/未命中/暴击/法术施放四种音效均可区分
- [ ] **AC-AUD.05**：音量设置为 0 时完全静音（包括骰子和战斗）
- [ ] **AC-AUD.06**：多个音效同时触发时不互相截断（如同时三个人攻击）
