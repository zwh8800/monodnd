## 1. 概述

### 1.1 文档目的

本文档定义《酒馆与命运》中**每一个屏幕、每一种交互、每一处视觉规格**的完整技术设计。它是美术师的像素资产参考、程序员的实现蓝图、设计师的交互规范。

### 1.2 视觉风格指南

#### 1.2.1 核心参考：SFC/FF6 像素艺术

本游戏的视觉风格严格遵循 SFC（Super Famicom）时代的像素艺术传统，核心参考为《最终幻想6》（FF6）和《超时空之轮》（Chrono Trigger）。

| 元素 | 规格 | 说明 |
|------|------|------|
| 角色精灵 | 16×24 px | 2头身比例，4方向行走，每方向3帧动画 |
| 地图Tile | 32×32 px | 5尺×5尺游戏内网格 |
| 窗口框架 | 深色镶边 + 半透明背景 | FF6风格暗色边框，内填充深蓝/深灰半透明 |
| 字体 | 8×8 px 像素字体 | 类SFC对话字体，支持中文需16×16 CJK字符集 |
| 调色板 | 暖色基调 + 暗色点缀 | 见§12完整色板定义 |

#### 1.2.2 窗口Chrome风格

所有UI窗口采用统一的FF6风格窗口框架：

```
窗口Chrome结构 (放大示意):
┌──────────────────────────────┐  ← 外边框: 2px 深色 (#1a1a2e)
│╔════════════════════════════╗│  ← 内边框: 1px 高光 (#4a4a6a)
│║                            ║│
│║    窗口内容区域              ║│  ← 背景: 半透明深色 (rgba(10,10,30,0.85))
│║                            ║│
│╚════════════════════════════╝│  ← 底部内边框: 1px 阴影 (#0a0a1a)
└──────────────────────────────┘  ← 底部外边框: 2px 深色

圆角: 无 (像素风格保持直角)
阴影: 窗口右下方2px偏移, rgba(0,0,0,0.5)
```

#### 1.2.3 核心设计原则

| 原则 | 说明 | 实施要求 |
|------|------|----------|
| **像素完美** | 所有UI元素对齐像素网格，无亚像素偏移 | 使用像素对齐，确保UI元素对齐像素网格 |
| **可读性优先** | 文字在任何背景下清晰可读 | 所有文字带1px深色描边或背景板 |
| **响应式缩放** | 支持1280×720到2560×1440 | 采用响应式缩放策略，基准分辨率1280×720 |
| **怀旧但功能** | SFC美学不牺牲现代UX | 保留hover提示、快捷键、工具提示 |
| **信息层级** | 重要信息突出，次要信息弱化 | 使用颜色/大小/动画区分层级 |
| **一致性** | 所有屏幕共享同一套UI组件库 | 统一Theme Resource |

### 1.3 目标分辨率与缩放

| 参数 | 值 | 说明 |
|------|-----|------|
| 设计基准分辨率 | 1280×720 (16:9) | 所有UI元素基于此分辨率设计 |
| 最小支持分辨率 | 1280×720 | 低于此分辨率不保证显示 |
| 最大支持分辨率 | 2560×1440 | 2×缩放，像素完美 |
| 缩放策略 | 响应式缩放: 基准分辨率1280×720自适应缩放 | 参考分辨率1280×720自适应缩放 |
| 缩放因子选项 | 0.8×, 1.0×, 1.2× | 玩家可在设置中调整 |
| 像素对齐 | 所有坐标取整到整数像素 | 避免模糊 |

### 2.2 屏幕转换类型

| 转换类型 | 时长 | 效果 | 使用场景 |
|----------|:---:|------|----------|
| `fade_black` | 0.3s | 画面渐黑→加载→渐亮 | 主要屏幕切换（酒馆↔冒险） |
| `slide_left` | 0.25s | 画面左滑出，新画面右滑入 | 酒馆内区域切换 |
| `slide_right` | 0.25s | 反向滑动 | 返回上一区域 |
| `overlay_pop` | 0.15s | 从中心弹出，带缩放 | 覆盖层（招募板/商店/对话） |
| `overlay_slide_up` | 0.2s | 从底部滑入 | 角色面板/物品详情 |
| `instant` | 0.0s | 立即切换 | 同屏幕内状态变化 |
| `dissolve` | 0.5s | 像素化溶解 | 记忆/闪回/魔法传送 |
| `dice_roll` | 1.0s | 骰子滚动动画 | 检定结果展示 |
